General

The Westminster Leningrad Codex (WLC) has been transcribed to files in the Books directory having Unicode Hebrew characters with XML markup. The displayed Tanach text is obtained from these formal electronic files. The files are formal in the following sense:
  • The provenance of the text is fully documented in each file's Text Encoding Initiative (TEI) header which can be seen by clicking on the page header.

  • The XML markup of each book file is validated with the W3C-Schema file, Tanach.xsd.

  • A secure hash (SHA-1) of each file is calculated on generation and checked periodically. These checksums are available from the Transcriber on request.
Additional technical details are given in the XML files page.

If permitted by your browser, this site will install a "cookie" on your machine to remember the setting of the pull down lists and text input fields. This cookie is not used to gather information about you. The site operates correctly, though awkwardly, if cookies are blocked.

When viewed as a remote web site, text is provided by the server Server.xml to reduce download volumes.

Operation without the internet:

A local version of the site can be installed on your computer for situations in which an internet connection isn't available. The local version will not include complete books with alternative contents and formats. It also will not include the Leningrad Codex facsimile files. To setup operation without the internet, download Tanach.zip (about 3.9 megabytes) into a convenient directory, say /MyDirectory. Unzip Tanach.zip into that directory so that a new directory /MyDirectory/Tanach of about 19.2 megabytes is created. Open /MyDirectory/Tanach/Tanach.xml with the Firefox or Safari browser to view the local site. The Internet Explorer and the Chrome browsers will not open this file.

The primary contents of the site can be viewed from the expanded Tanach.zip. Text, ODT, and PDF files are not included in Tanach.zip. A secure (SHA1) hash for Tanach.zip and other files can be provided on request.

Determining the version of the site:

The build number at the bottom right of each page consists of a major version number, a decimal point (.), and a minor version number ). Changes in major version are described on the Changes page. Minor version changes are undocumented. They usually consist of corrections imperceptible to the average user. The version of the WLC appears in square brackets on text pages. Differences between WLC versions are documented.

The "Previous versions" link in the upper, left corner of the Home page allows access to recent versions of the site. A historical perspective of all versions is available.

Software:

Eight Java applications that are part of the site are provided in the Java subdirectory:

DHMarker Applies Documentary Hypothesis markings to the Torah XML files.
HTMLServer Java servlet Server.html which serves user-formattable HTML text.
MC2U Java application to convert short MC-coded strings to Unicode/XML text. In the Supplements directory.
ODTServer Java servlet Server.odt which serves ODT files.
Supplements Generates the supplementary files in the Supplements subdirectory from the XML files.
TextServer Java servlet Server.txt which serves text files.
WLC2XML Transcribes the Michigan-Claremont encoded J. Alan Groves Center for Advanced Biblical Research text to Unicode with XML tags.
XMLServer Java servlet Server.xml which serves XML text.

Source and class files for each application are given in a Java archive (jar) file. Contact the transcriber if you need assistance in using them.
