About

This application is an adapted version of the http://www.tanach.us/ web site (particularly of the offline or local site).

It's using Crosswalk 20 (Chromium M50) with the aim (and hope) to display correctly the character set of SBL Hebrew font.

I appreciate your time and interest in using this application. Please feel free to send your feedback/suggestions for any improvements to support@paulvargas.org.
