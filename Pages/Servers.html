Servers

The tanach.us site offers many features and imposes the necessary bandwidth, memory, and computing resources on the user's device.

A faster and lighter way to obtain Hebrew text from the site is with one of the 4 servers. These servers produce XML, text, HTML, or ODT formatted content depending on the particular server called. Each server requires a valid text citation possibly followed by additional parameters. A valid citation syntax for the servers has one of three formats defined on the Citation syntax page:

  • Chapter:Verse range: b c1:v1 - c2:v2
  • Single chapter:Verse range: b c:v1 - v2
  • Complete book: b *
Other formats are invalid for the servers. Some servers allow URL parameters to customize the display.

A simple example of a server call is:

http://www.tanach.us/Server.html?Deut26:5-9
which shows Deuteronomy 26:5-9 in HTML from the Server.html server. When this URL is entered into a browser, the only downloaded content is the HTML file itself; the tanach.us site does not need to be loaded at all. Incorrect citations or parameters produce meaningful error messages:
http://www.tanach.us/Server.html .

Click on the links in the "Server" column for more information about a particular server.

Checking operation of the servers.

The status of the servers can be checked by the URL:
http://www.tanach.us/ServerTest
If the servers are not operational, contact the Transcriber for repairs.
