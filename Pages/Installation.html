Installation

Because the accent markings aren't required by modern Hebrew and have little commercial impact, the fonts provided with computer systems are usually inadequate for complete displays of the Masoretic text. Hence some care in installing the site is necessary for best operation.

  • Users of Mac and Windows operating systems should download and install the Society of Biblical Literature (SBL)  SBL Hebrew font, SBL_Hbrw.ttf, making note of the License agreement. It is the default font for the site.

  • The built-in Mac Raanana font is also satisfactory, but needs to be set into the site via the "Font" pulldown list "Change font" item on any Book page.

  • Users of Linux operating systems should download and install the Culmus Project Frank Ruehl CLM font. It must be set into the site via the "Font" pulldown list "Change font" item on any Book page.
The font is saved as a cookie and needs to be set only once.

The FontTest page compares the rendering on your machine with images of properly rendered text and allows changing the Hebrew font.

On any operating system the free Mozilla Firefox browser, is highly recommended.
