MC2U: Michigan-Claremont text to Unicode converter

Your browser lacks the required Java plugin.

Install the Java plugin or download the MC2U.jar file for offline operation. The Java runtime environment (JRE) is available for free!

This Java applet will put files in the indicated output directory when the "Convert to Unicode" button is pressed. All these files start with MC2U and may be deleted at any time. The text displayed initially in the text field is for demonstration purposes and corresponds to Genesis 1:1.
This application is available as a Java jar file, MC2U.jar. In some cases, operation is easier if MC2U.jar is downloaded and run directly. The Java Runtime Environment (JRE) is still required, however.
